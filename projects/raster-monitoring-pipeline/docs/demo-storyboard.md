# Demo Storyboard

Reference artifact: a real raster-difference image, hotspot map, or other generated output.

## 1. Frame the use case

Show the project as a lightweight raster change-detection lane for environmental monitoring.

## 2. Explain the inputs

Point out that the repository includes baseline and latest raster-style grid fixtures.

## 3. Run the pipeline

Generate `outputs/raster_change_report.json` and show the hotspot counts, top changes, and tile manifest.

## 4. Explain the extension path

Close by noting that the same structure can grow into GeoTIFF-backed workflows through rasterio or GDAL.
