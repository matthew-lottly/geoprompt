CausalLens JSS Upload Bundle

This folder contains the concrete files to upload or attach for the JSS submission review package.

Included files
- causal-lens.pdf: manuscript PDF in JSS format
- causal-lens.tex: LaTeX manuscript source
- causal-lens.bib: BibTeX references
- jss.cls: JSS class file
- jss.bst: JSS bibliography style
- jsslogo.jpg: JSS logo asset used by the template
- causal-lens-source.zip: source-code snapshot from the tracked repository state
- replication-output.log: saved console output from running replications/run_all.py
- LICENSE: MIT license (GPL-compatible for JSS)
- CITATION.cff: citation metadata

Reviewer notes
- The Word file in docs/causal-lens-review-copy.docx is for author review only and is not part of the submission bundle.
- The manuscript source under .private-drafts is gitignored, so the copies in this folder are the submission copies.
- The replication log was generated from the standalone workflow using the current project environment.

Recommended submission contents
1. Upload the manuscript PDF.
2. Attach the LaTeX/BibTeX source and JSS style files.
3. Attach the source snapshot and replication log.
4. If the submission portal limits file count, upload this folder as the zip archive created alongside it.
