# Suggested Public Issues

## 1. Add time-window filters to observation history

Allow callers to request observations within the last 24 hours, 7 days, or a custom start and end range.

## 2. Surface recent observations in the dashboard

Add a dashboard panel that highlights the latest alert observations and recent status changes for monitored stations.

## 3. Add threshold configuration per station

Model station-specific thresholds so alert status can be derived from observation values instead of stored directly on the station record.
