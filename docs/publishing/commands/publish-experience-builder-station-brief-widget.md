# Publish Commands: Experience Builder Station Brief Widget

Create an empty GitHub repository named `experience-builder-station-brief-widget`, then run:

```powershell
Set-Location d:\GitHub\standalone-checkouts\experience-builder-station-brief-widget
git init
git add .
git commit -m "Initial standalone release"
git branch -M main
git remote add origin https://github.com/matthew-lottly/experience-builder-station-brief-widget.git
git push -u origin main
```

If the repository already has a remote configured, skip the `git remote add origin ...` line.
