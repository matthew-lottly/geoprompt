# Suggested Public Issues

## 1. Publish a container image from CI

Extend the CI pipeline so pushes to the main branch build and publish a container image for the standalone API repository.

## 2. Add threshold configuration per station

Model station-specific thresholds so alert status can be derived from observation values instead of stored directly on the station record.

## 3. Add export-ready operations summaries

Add a compact endpoint or generated artifact that summarizes alert pressure, recent changes, and key operational metrics for downstream reporting or handoff.
