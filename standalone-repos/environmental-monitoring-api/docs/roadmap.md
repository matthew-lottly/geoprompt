# Roadmap

## Near Term

- Expand observation-history support with time-window filters and category-level rollups.
- Add dashboard panels for recent observations and trend changes.
- Publish a container image as part of CI.

## Next

- Add authenticated write endpoints for ingestion from upstream monitoring feeds.
- Add station configuration and threshold management.
- Add export-ready summaries for operations teams.

## Longer Term

- Add alert rules that derive status from observation thresholds instead of seeded station states.
- Add time-series storage and aggregation patterns for larger observation volumes.
- Add role-based access patterns for field teams and analysts.
